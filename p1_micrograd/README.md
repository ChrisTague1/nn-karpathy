# Setup

- Will need to install [graphviz](https://pypi.org/project/graphviz/)

# Kernels

To register a kernel:

```bash
python -m ipykernel install --user --name=myenv
```

To list kernels:

```bash
jupyter kernelspec list
```

To delete kernels:

```bash
jupyter kernelspec remove kernel_name
```
